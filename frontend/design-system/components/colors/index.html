On accessibility. Brand sky fails WCAG AA for small text (about 2.9 to 1 on white) so it is an identity color only, used for the BeakerBot stroke, large marks, and gradients, never body copy. Brand action is the accessible UI accent (about 4.5 to 1) and is what text, links, and buttons use. Accent and brand action are the same value by design.
