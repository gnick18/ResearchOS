# [Demo protocol] Growth curve in 96-well

Reads OD600 every 15 min for 18 h to compare strain doubling times.

## Layout

- Rows A–D: wild-type `FakeYeast-001`, 4 biological replicates.
- Rows E–H: engineered `FY-Δgal80`, 4 biological replicates.
- Each row gets four glucose concentrations across columns 1–12.

## Steps

1. Inoculate overnight cultures in YPD; back-dilute to OD600 ~0.05.
2. Load 200 µL per well, cover with breathable seal.
3. Plate reader: 30 °C, double-orbital shaking, OD600 every 15 min, 18 h.
4. Export the kinetic dataset as CSV; the `Growth curve` template at `users/public` parses it directly.

## Common gotchas

- Use clear-bottom black-walled plates; edge wells dehydrate first, so leave A1 and H12 empty if you can.
- Subtract the well's t=0 reading before fitting doubling time.
