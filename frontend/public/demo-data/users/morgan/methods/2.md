# [Demo protocol] qPCR setup

Two-step RT-qPCR for `fakeGFP` transcripts with ACT1 reference.

## Reagents

- SYBR Master Mix (2x)
- 10 µM `fakeGFP-fwd` / `fakeGFP-rev`
- 10 µM `ACT1-fwd` / `ACT1-rev`
- cDNA, diluted 1:5 in nuclease-free water

## Cycling

- 95 °C, 3 min initial denaturation
- 40 cycles: 95 °C 15 s → 60 °C 60 s (anneal + extend)
- Default melt-curve

## Layout

- Run targets and reference in adjacent wells per sample to keep block-position noise paired.
- Always include a no-RT control per template; if anything amplifies before cycle 35 the sample is contaminated.
