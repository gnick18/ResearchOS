# [Demo protocol] Fluorescence screen (96-well)

Endpoint GFP read for screening `FY-Δgal80` transformants.

## Reader settings

- Excitation: 485 nm.
- Emission: 528 nm.
- Gain: 60 (adjust if positive control saturates).

## Layout

- Column 1: WT (no GFP)
- Column 12: positive control (pDEMO-fluo+)
- Columns 2–11: candidate transformants, one per well.

## Steps

1. Pick colonies into 200 µL SD-His + 2% galactose, grow 16 h at 30 °C.
2. Read OD600 *and* GFP.
3. Normalize per-well GFP by OD600, then divide by column-12 mean.
4. Anything ≥0.6× of the positive control is a hit.
