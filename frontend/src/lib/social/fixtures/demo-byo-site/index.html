Abstract

This is a fully custom static site the lab built itself and hosts on its own slug. It is the same paper as the native companion page, shown here as a bring your own bundle so a prospective lab head can see that one lab can publish both a ResearchOS authored site and its own static site from a single handle.

Interactive figure

A real paper companion would mount its own charts and widgets here. The point of the demo is the hosting path, served from a separate sandboxed domain so the lab JavaScript stays isolated from the authed app.
