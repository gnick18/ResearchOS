# ResearchOS frontend

This folder is the Next.js app. The project root has the actual README, including run-it-yourself instructions, the feature overview, and links to the in-app wiki.

See [../README.md](../README.md).

## Local dev shortcut

If you're already inside this folder:

```bash
npm install
npm run dev
```

The dev server runs at http://localhost:3000. For everything else (deployment, browser support, security model, contribution guide), use the root README.
