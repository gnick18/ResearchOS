What changed, and why

  1. 1 Operations rail: every bioinformatics tool is a visible icon (no junk drawer). Glanceable, opens inline panels. (Geneious / UGENE)
  2. 2 Contextual inspector: select a feature/region and the tools for it surface automatically. The biggest "unburial" move. (Figma / NN/G)
  3. 3 Display toggles split from operations: "what is drawn" lives in the canvas strip, not mixed with "run a tool." (SnapGene)
  4. 4 One live canvas, many lenses via tabs (Map / Sequence / Features / Primers / History).
  5. 5 Your right-click system is kept everywhere as the object-scoped accelerator that duplicates these visible actions.
Tip: click the rail icons; toggle the Tree (amber dot = this sequence has an organism). Hit ⌘K for the palette. Toggle these notes from the top bar.
