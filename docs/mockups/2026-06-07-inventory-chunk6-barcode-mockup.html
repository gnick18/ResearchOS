Chunk 6 adds scanning on /inventory. You scan a lab-applied container code to mark one used (count down) or a manufacturer barcode to register an item. Per the research, the lab-applied code is the primary path; the manufacturer-lookup is an optional bring-your-own-key bonus. Built for the desktop webcam first; the phone reuses the same logic later.
