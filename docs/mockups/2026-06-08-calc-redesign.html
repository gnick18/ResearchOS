Calculator redesign, for review

The Scientific tab gets a clean phone-calculator layout: a dark result display, the scientific functions as a single scrollable strip (no more cramped twin grid), and a roomy four-column pad with sky operators. Below that, three options for where the calculator-switcher (Scientific / Molarity / Dilution / Serial / Buffer) lives, since the current top chip row felt off. Tell me which switcher you want and whether the keypad direction is right.
