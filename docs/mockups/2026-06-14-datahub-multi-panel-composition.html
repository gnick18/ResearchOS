Multi-panel figure composition

Arrange several figures as Figure 1A / 1B / 1C on one publication page, annotate, export as one exact SVG.

The artboard already puts one figure on a real publication page at true scale. This extends it to many: drop several figures on the same page, drag them into a panel layout, and they auto-label A / B / C. Add a freeform annotation layer (text, arrows, lines, brackets, significance markers) over and between the panels. Export the whole page as one publication-exact SVG. It is the same shared lib/figure artboard, so Tree Studio gets it too (compose a phylogeny + its data panels on one page).

Try it: click + Add figure, drag the panels to arrange them, click a panel to select + resize it, pick an annotation tool and click the page to drop one. Flip the panel-label style. Then mark the decisions below.
