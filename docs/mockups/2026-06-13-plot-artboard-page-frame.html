Plot artboard / page frame

Design the figure inside a real publication page, so the exported SVG fits perfectly.

The idea: an optional page frame behind the plot (like an Illustrator artboard). You pick a real paper or journal-column size, and the figure sits inside it at true scale, so you can see at a glance whether you have room to go bigger or need to shrink. Because the figure is sized in real units, the exported SVG is publication-exact, no guessing. This is a shared canvas capability for every plot editor (Tree Studio, Data Hub plots, anything with a figure).

Try it: change the paper, flip orientation, drag Figure size, hit Fit. Watch the feedback chip and the export readout. Then mark the decisions below.
