Phase 2 + 3 UI, 2026-06-13. The "edit with code" builder and the wide-column manager. Mark up and Export.

What you are reviewing

Two surfaces. The transform builder is the fleshed-out "edit with code" tool. It is the Data Hub's existing pandas-matched transform engine, given a full operation menu and a live code preview, running in the background engine on the whole dataset. The wide-column manager is how the preview behaves when a table has dozens, hundreds, or thousands of columns. Add a few operations, switch the column tier, and feel both.
