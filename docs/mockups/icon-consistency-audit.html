Website-wide icon consistency audit

Every icon on a button, tab, menu, list-row action, or nav is a custom inline SVG (house rule, no icon library). This page hunts the same action drawn DIFFERENT ways across the app, plus stale glyphs that should adopt the set we just standardized in the sequence editor.

Each card shows the exact SVG from the source, rendered on a light and a dark swatch, with the files it lives in and the label it sits next to. The first card in a group is the CANONICAL one to keep. STALE means replace it with the canonical. NEW means a glyph or concept no reviewed set has approved yet, so it needs a decision.

Decisions live up top. Groups with drift and the NEW unreviewed glyphs come first. The already-consistent groups are at the bottom for confirmation.
