Phase 1 mockup · icons in figures. A searchable library of NIH BioArt vectors drops recolorable icons onto a publication page, with credits auto-generated. Representative icons shown; real ingest pending.

Only the legally-clean subset is ingested. We keep Public Domain CC-BY CC-BY-SA and hard-exclude CC-BY-NC CC-BY-ND (NC = no commercial use for a paid product; ND = we recolor/resize, which is a derivative). CC-BY icons are cited automatically; Public Domain needs no citation.

What this shows. An icon is a lightweight, recolorable graphic placed on the page — not a data "panel." It reuses the picker (search + category chips), the Phase-3 recolor seam (the Color row), and the exact-SVG compositor (so on-screen == export).   Auto-credits are the legal load-bearing piece: every CC-BY icon contributes a verbatim citation to a "Figure credits" block, so users are compliant by construction — Public Domain icons add nothing.   Next: Phase 0 ingest (IDs 1..~2200, keep PD + CC-BY + CC-BY-SA), then this Phase 1 surface, then a freeform diagram canvas and poster paper presets.
