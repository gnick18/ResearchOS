A transform glyph for the Data Hub

A derived table (one built from a recipe of transforms, like filter, join, or derive) carries a small badge in the navigator, and the Transform dialog uses the same glyph. The placeholder reused the merge icon (the git-branch shape), which reads as version control, not data transformation. Drawn in the house style (viewBox 24, stroke-width 2, round caps, no fill, currentColor), the same as the rest of the registry. Pick one, tweak, or reject all. Toggle dark to check both themes, and remember the navigator badge renders tiny (about 10px), so legibility at the 16px size matters most.

The badge shows up small and accent-sky in the rail. Each card has a live "rail badge" preview at the true rendered size so you can judge it where it actually appears, not just at the comfortable large size.
