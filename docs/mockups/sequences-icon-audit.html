Sequences page icon audit

Every icon below is the exact SVG from the code. Mark the ones that do not read.

Grouped by where it appears in the editor. An icon reused in several spots is listed once, with all its homes noted. Amber pills flag likely confusable pairs.

Look for: icons that say nothing on their own, two glyphs that look alike, and a glyph whose shape fights its meaning.

Top action bar

The document-editing row at the very top of the editor (hidden in read-only / embedded). Undo, Redo, the Copy split button, Cut, Paste, Save.

Undo IconUndo
Label: "Undo" / tooltip "Undo (Cmd+Z)"
Step the document back one edit. Left-curving return arrow.
Redo IconRedo
Label: "Redo" / tooltip "Redo (Cmd+Shift+Z)"
Step the document forward one edit. Right-curving return arrow (mirror of Undo).
Copy IconCopy
Label: "Copy" split button / tooltip "Copy (Cmd+C)"
Copy the selected bases. Two overlapping sheets.
looks like: ActionGlyphs.copy (inspector), IconDuplicate (primer dialog) — same two-sheets shape
Cut IconCut
Label: "Cut" / tooltip "Cut (Cmd+X)"
Remove the selected bases to the clipboard. Open scissors.
looks like: IconEnzymes (Show strip "Enzyme sites") — both are scissors; nearly identical paths
Paste IconPasteTool
Label: "Paste" / tooltip "Paste annotated sequence (Cmd+V)"
Insert clipboard bases at the caret. Clipboard tool.
looks like: ActionGlyphs.paste (inspector) — a different clipboard glyph for the same idea
Save IconSave
Label: "Save" / "Saving…" / "Saved" / tooltip "Save (Cmd+S)"
Write the edited document to disk. Floppy disk.
Split-copy caret Chevron (SequenceEditMenu)
Label: none. The caret on the right half of the Copy split button (and on the menu dropdowns).
Open the full copy family menu (bottom strand, amino acids, FASTA, map image). Down chevron.
looks like: ChevronDownIcon (Import), IconCaret (Show strip), Chevron (Export), CaretIcon (lineage) — five separate down-chevron defs

Right rail (operations)

The thin always-visible icon strip on the right edge. Each icon + its tiny label opens an inspector panel. Grouped Design then Analyze, then Export + More.

Primers RailIcons.primers
Label: "Primers"
Primer design / check tools. A line with a forward arrowhead (an oligo annealing 5' to 3').
looks like: IconPrimers (View tabs), IconOrfs (Show strip) — all arrow-on-a-line glyphs
Cloning RailIcons.cloning
Label: "Cloning"
Assembly / cloning chemistries. Two circles joined by a link.
looks like: RailIcons.tree / ActionGlyphs.tree / lineage TreeGlyph — all "circles joined by lines" three-or-two-node shapes
Cut (rail) RailIcons.cut
Label: "Cut" (restriction-enzyme panel)
Restriction digest tools. Scissors (closed X blades).
looks like: IconCut (top bar), IconEnzymes (Show strip) — three scissors glyphs for two different meanings (edit-cut vs enzyme-cut)
Annotate RailIcons.annotate
Label: "Annotate"
Add / detect features. A bare checkmark.
looks like: ActionGlyphs.check, the toast success check — a plain check that reads as "done", not "annotate"
Align RailIcons.align
Label: "Align"
Compare / align sequences. Three ragged-length lines (like left-aligned text rows).
looks like: ActionGlyphs.list, ActionGlyphs.align (identical path), IconWrapped (Show strip) — four near-identical "stacked lines" glyphs
Protein RailIcons.protein
Label: "Protein"
Protein properties of a coding feature. Two crossing curves (an X of helices).
looks like: ActionGlyphs.protein (identical path)
Tree RailIcons.tree
Label: "Tree" (taxonomy / lineage)
Tree of life / taxonomy explorer. Three nodes joined (two on the left, one on the right).
looks like: ActionGlyphs.tree (identical), lineage TreeGlyph, list-row organism glyph, RailIcons.cloning — the most-reused node-graph shape
Export RailIcons.export
Label: "Export"
Save the sequence out as a file. Down arrow onto a baseline.
looks like: ActionGlyphs.download (identical), NcbiCloudIcon down-arrow, IconExport (Export menu) — three download-style arrows
More RailIcons.more
Label: "More"
Overflow / additional operations. Three dots.
