Mobile cosmetics pass

Screen-by-screen aesthetic fine-tuning of the Expo-Go mobile app, captured live on the Android emulator and checked against the approved apple-polish ideal. Four UI-only commits, all landed on mobile main. The signature trio is sprinkled per screen, never a full rainbow.

sky #1AA0E6 coral #FF6F61 amber #F59E0B

1. Near-white grey canvas

8aad62cd7

The old page was a sky-tinted brand blue (#E6F4FE). Swapped to iOS-style near-white grey (#f2f3f7) across every canvas source so the white popups stand out against the page instead of blending into it. Shown here as the current Notebook home, grey page with the trio on the action cards.

After, grey canvas + trio action cards

2. White sheets pop off the grey

183432007

Inverse-sheet pattern. The notebook chooser sheet went from grey (same hue as the page, leaning on the scrim to separate) to a pure-white sheet with grey-inset card groups, so the modal pops harder off the grey page. Recommended rows are sky-tinted.

Before, grey sheet
After, white sheet + grey-inset groups

3. BeakerBot warms the pairing screen

9620e2472

The pre-pairing screen was cold and mascot-less. Added a living BeakerBot above the title to match the approved Connect frame, so the first screen a new user sees is friendly and on-brand. The global rainbow bar caps the top.

Before, no mascot
After, BeakerBot anchors it

4. Running-timer trio accent

9be548b6e

The running countdown was all-blue, the one genuine trio gap found in the pass. Now an amber countdown with a coral Cancel, while idle presets stay blue. The Timer screen below shows the trio in its resting state.
