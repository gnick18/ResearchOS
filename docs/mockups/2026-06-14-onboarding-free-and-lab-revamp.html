Key idea. The sign-in popup and folder popup are independent reusable components. Local skips sign-in entirely. Free chains both popups in-place over the chooser without any navigation. Lab is the only tier that navigates to its own page, but it still recycles the same two popups within that page rather than inventing new ones.

Layout intent. The tile grid stays visible and blurs behind the overlay so users understand exactly where they are in the flow. Both popups use the same CSS scale + opacity spring easing (cubic-bezier .34 1.42 .6 1) so they feel like a matched pair. The sign-in popup has a back arrow and an X; the folder popup mirrors the same affordances. When the folder popup is reached from the FREE path the back arrow returns to the sign-in popup; when it is reached from the LOCAL path directly the back arrow is hidden (there is no prior step) and only the X is shown, so neither path is a soft-lock. The folder drop zone highlights on hover and drag-over; the "Simulate drag-over" button lets reviewers inspect that state without needing a real drag. ORCID gets its own row with a light green border to distinguish it from the big-tech row without demoting it to a footnote.

Create a lab

You become the lab head. You invite members, manage the workspace, and hold the lab's billing. Every member's data stays on their own disk; the cloud is only an intermediary for team sync.

  • Real-time co-editing for the whole team
  • PI oversight dashboard and check-ins
  • Shared methods, protocols, and inventory
  • Server-blind: encrypted before it leaves any machine
  • Free to create (1 GB pooled per member included)

You will sign in with a provider on the next step. Lab creation is free.

Layout intent. The left/right halves have equal width and equal visual start-weight so neither path looks like the "real" one and neither looks like an afterthought. The invite-link input is exposed immediately (no "have an invite link?" chevron to expand first) because users who arrive with a link should be able to paste and go with zero extra clicks. The directory search is a secondary affordance below the fold of the left half so it does not crowd the primary path. The right half has a subtle directional gradient to give Create a slightly elevated visual tier matching that it is the "starting something new" path.
