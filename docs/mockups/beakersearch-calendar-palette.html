Before (today)

The Calendar page registers no source, so Cmd-K shows only the always-on global layer. No context, no event actions, no jump-to, no Next up.

After (proposed Calendar source)

The page contributes a context card (the live frame + selection), selection-aware actions, jump-to-event and jump-to-date, a Next up list, and the full Create / Navigate / View / Feeds command set.

What the Calendar source adds. Mark each Approve or Disapprove.
