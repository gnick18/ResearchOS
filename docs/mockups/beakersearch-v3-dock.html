The chosen model

Floating dock that hides off the edge
  1. Opens as a small floating window you drag anywhere. It never resizes.
  2. Collapse button shrinks it to a pill; click the pill to reopen.
  3. Drag it to a side (or hit the hide button) and it slides off that edge, leaving a peek tab.
  4. Pull the tab back in for a re-check; the box returns at its same size.
  5. Close and Escape fully dismiss it; reopen returns it where you left it.

Try it

Drag the dock header toward the right or left side — a "Hide here" zone lights up and it slides off that edge into a tab. Pull the tab back in to re-check. The hide button (⇥) does the same without dragging. Cmd K toggles, R re-checks, esc closes.
