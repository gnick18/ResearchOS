One-way ANOVA

Comparing cell viability across Control, Drug A, and Drug B.

The three groups are not the same. Cell viability differs significantly across Control, Drug A, and Drug B (one-way ANOVA, F(2, 15) = 119.4, p < 0.0001). In plain terms, both drugs lower viability, and they differ from each other too. See the Tukey comparisons for which pairs differ. ${tagControls('plainlang','Plain-language results summary')}

from scipy import stats
import statsmodels.stats.multicomp as mc

control = [98, 102, 95, 105, 100, 99]
drug_a  = [78, 82, 75, 80, 85, 79]
drug_b  = [55, 60, 52, 58, 50, 57]

F, p = stats.f_oneway(control, drug_a, drug_b)
# F = 119.4, p = 2.8e-10

tukey = mc.pairwise_tukeyhsd(
    endog=control + drug_a + drug_b,
    groups=["Control"]*6 + ["Drug A"]*6 + ["Drug B"]*6,
)
print(tukey)
