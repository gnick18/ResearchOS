Consistent title size + position, and three right-side buttons everywhere: Notifications, Today, Settings. Today now opens from its own header button (with a count badge), so the inline pill leaves the body. The connection status moves out of the busy top bar into a slim banner that appears under the header only when something needs you to know (offline / laptop asleep / queued), and is hidden when synced. Switch the tab + cycle the connection state to see it.
