Tree Studio: ggtree-class aligned panels TARGET

The vision: the tree is a shared tip axis, and data panels align to it tip-for-tip, the way ggtreeExtra::geom_fruit stacks rings and aplot links plots. Panels render with continuous color scales and legends, powered by the Data Hub plot engine. Toggle "Today vs Target" to contrast with the current iTOL-basic Studio. This is a design mockup, not the live app.
