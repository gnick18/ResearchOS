The proposed flow

A note often embeds molecules, sequences, Data Hub results, or other notes. Today sharing a note across labs sends only the text, so the embeds land as dead links for the recipient. The flow below bundles the note and what it references into one sealed package, lets the sender see and trim the dependencies, and lets the recipient choose where each item lands. Same-lab shares are unchanged (embeds already resolve natively).
