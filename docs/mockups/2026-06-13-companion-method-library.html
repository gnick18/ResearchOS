How it works

  1. Active recs on top. Methods on your ongoing experiments float to a highlighted band with a live Active dot, so the thing you are running is one tap away. The full library follows.
  2. Search + filter + sort. Live search, horizontally scrolling type chips (PCR / LC-MS / Cloning / ...), and a Type/Recent/A-Z sort. Built for hundreds.
  3. Offline download. On pair, the app asks once to download the library. The status chip shows offline-ready, and turns amber Update when the laptop changes a method.
  4. Grouped + scannable. Sticky section headers, type-colored icons, favorites star, "on this phone" check.
  5. Tap a method to open the big-text read mode from the other mockup.
Type "pcr" or "extract", tap a filter chip, or tap the green chip to cycle offline states.
