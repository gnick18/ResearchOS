Before (today)

The Lab Overview registers no source, so Cmd-K shows only the always-on global layer. No lab snapshot, no inline approvals, no jump-to a member or a pending item.

After (proposed Lab Overview source)

A context card (lab size + the inbox snapshot), INLINE approve / decline / assign / flag via the per-record PI confirm, Post an announcement, jump to any member / pending approval / flagged record / section, and the route-outs to Purchases / the lab inbox.

Note, this page changed since the spec. The PI capability revamp rebuilt Lab Overview into a curated hub, the old timed edit-session (unlock / lock, "4:12 left", password modal) is GONE. Announcements are author-gated, never session-gated.
