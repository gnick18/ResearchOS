Translation amino-acid box, tip options

Each amino acid is drawn as a box with a directional tip. Today it is a fixed sharp chevron with a notched left edge that interlocks (option A). Here are cleaner treatments at the same zoom. Two AA colors shown so you can see how the tips read in a run. Pick a letter.

A. Current, sharp interlocking chevron

Fixed 3px point on the right, a matching 3px notch on the left so boxes interlock. The fixed size reads sharp/jagged when zoomed.

B. Subtle chevron, proportionalrecommended

A gentler tip scaled to the box height, still interlocking and directional, just calmer.

C. Flat rounded boxes, one arrowhead at the run end

Each amino acid is a clean rounded rectangle; a single arrowhead caps the whole run to show direction. Calmest, most SnapGene-like.

D. Rounded chevron tip

Same interlocking chevron as A but with softened (rounded) points instead of hard corners.
