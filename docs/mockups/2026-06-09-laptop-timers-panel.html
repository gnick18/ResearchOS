Laptop timers panel

Phase 3, cross-device timer sync. The laptop gets a real two-way timers panel that stays in lockstep with the phone.
ApproveChange?

No laptop timer exists today, so this is the net-new half of the integrations work (the phone timers + alarm already shipped). The design mirrors the phone screen in the web app's language and adds the cross-device sync the whole feature is about. A timer set on either device exists on both, ticks locally on each (no network per second, just an absolute end time), and syncs only its create / done / dismiss events through the relay. Silencing on either device silences both. Per the locked decisions, the laptop gets a full create / list / cancel panel (not a fire-and-dismiss-only alarm), and each device keeps its own alarm settings with the full Chime experience as the default everywhere.
