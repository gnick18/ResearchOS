Before (today)

The Workbench registers no source, so Cmd-K shows only the always-on global layer. No tab context, no per-tab actions, no cross-tab jump-to-anything.

After (proposed Workbench source)

One source spanning all five tabs, a context card (tab + filter + selection), selection-aware actions, and the headline, reach any project / experiment / list / note / notebook / 1:1 from any tab, plus tab switches and Recently opened.
