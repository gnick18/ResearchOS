This is the Connect tab when you have no account yet. Today it is a flat blocker ("Make an account to use Companion" plus a keypair explanation and the dev buttons), which reads as a wall, not an invitation. The companion is optional and free to run (phone push costs us nothing), so this gate is purely an acquisition funnel, getting a researcher to make a free account, the on-ramp to the optional cloud sync and AI tiers. The redesign briefly sells the phone companion with the real shipped capabilities, then frames Create a free account as the unlock, with one honest line that the same account also opens optional cloud sync and the AI assistant. The dev buttons stay, demoted into a disclosure. Toggle dark mode to check both themes.
