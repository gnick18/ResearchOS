What the proposed adds (all driven by real state):

1. On this sequence a context card, name, type, topology, length, features, organism, and the live selection (with Tm/GC), so you always know what you are acting on.
2. Suggested the selection-aware commands, with the selection echoed in the row (from 612..632, 21 nt) so it reads as "do this to my highlight."
3. Jump to a sequence the other sequences in the current collection, so BeakerSearch also navigates, not just runs tools (this is the one genuinely new capability, it needs the palette to reach the sequence list).
4. Recent results the latest saved artifacts (Phase 5), reopenable from here.
Once you type, the context card stays as a slim header and everything below collapses to ranked fuzzy matches across all four kinds (commands, sequences, results).
