Built on open source

We stand on a lot of other people's work

Every screen rests on open-source software written by volunteers, students, and engineers who chose to share. Several licenses also require attribution, so this is both a thank-you and an obligation we take seriously.

The writing surface

CodeMirror, marked, the unified remark/rehype pipeline, turndown

Sequence and cloning

Konva and react-konva, plus vendored SeqViz and TeselaGen bio-parsers

Calculators and math

expr-eval-fork, and a primer Tm ported from Biopython

Charts, files, and state

Recharts, frappe-gantt, JSZip, Zustand, TanStack Query, date-fns

The framework

React and Next.js

Code we recycle

SeqViz (MIT), TeselaGen bio-parsers (MIT), the Biopython Tm port (BSD), carried with their licenses
The full per-package license inventory lives on the open source page and in THIRD_PARTY_NOTICES, generated straight from the installed dependency tree so it never drifts.
