# ResearchOS v0.3.0

Released 2026-06-03 (beta channel)

This is the largest release since the dashboard rebuild. The headline is a brand-new Sequence Editor: a built-in, SnapGene and Benchling style molecular-biology workspace that lives right alongside your projects, notes, and methods. Around it, the whole app got calmer and more focused: the home dashboard is now a curated page instead of a customizable widget grid, Projects open by default, Experiments became a clean pipeline board, and PIs get a dedicated Lab Overview. There are also new lab tools, a security upgrade, and an open-source credits page.

ResearchOS stays free and open for every lab, runs entirely on your own machine against your own data folder, and ships under AGPLv3.

## Sequence Editor (new)

A full molecular-biology surface for your plasmids and DNA, reachable from its own top-nav entry and from any sequence attached to a project.

- **Import what you already have.** Open SnapGene `.dna` files, GenBank (`.gb`/`.gbk`), and FASTA directly. The reader handles modern SnapGene exports, and your features, colors, notes, and designed primers all come across.
- **One fluid editor, no modes.** There is no separate read versus edit toggle (SnapGene and Benchling do not have one). You select, edit, and annotate in a single calm surface, with a live readout of coordinates, base count, and GC content for any selection.
- **SnapGene-style navigation.** A bottom tab bar (Map, Sequence, Enzymes, Features, Primers, History) switches views the way you expect. A persistent overview strip shows the whole molecule with a viewport box that shrinks and slides as you zoom and scroll, alongside a zoom slider, an editable bases-in-view field, an exact position readout, and a draggable coordinate scrollbar. Pinch to zoom on a trackpad.
- **Annotations you control.** Add, edit, recolor, and show or hide features by type from an always-accessible icon rail. The feature editor includes a mini gene-viewer diagram with strand-direction arrows and multi-segment support.
- **Genes with introns, done right.** Multi-exon (spliced) genes render as connected exon boxes joined by dashed intron lines with a single label, and their translation is correctly spliced, so the protein reflects the joined exons rather than reading straight through the introns. Overlapping gene/mRNA/CDS annotations no longer paint the same protein multiple times.
- **Primer design.** Design primers with melting temperatures and an alignment preview popup. Melting temperatures use the nearest-neighbor model and match the lab calculator.
- **Restriction enzymes.** Pick from commercial enzymes and see exactly where each one cuts, with filtering and a digest summary.
- **Editing and clipboard parity.** Right-click menu, keyboard shortcuts, an Edit menu, annotated copy and paste between sequences, copy of the bottom strand or translated protein, select-all/range/invert, case changes, find, and go-to.
- **Export.** One click to GenBank, FASTA, the current selection as DNA or translated protein, or the map as an image.
- **Per-project sequences.** Sequences attach to projects as their own tab, so each project carries its own constructs.

## A calmer, more focused workspace

The minimalism arc removed clutter across the app without removing capability.

- **Curated home dashboard.** The customizable widget grid is retired in favor of a fixed, well-organized home so nothing has to be assembled before it is useful.
- **Projects open by default**, giving each project real standalone utility.
- **Experiments as a pipeline board.** The Experiments tab is now a compact kanban with a left accent, tidy chips, and capped columns.
- **Quieter chrome.** Calmer task popups, a quieted Telegram status badge, data-folder and switch-user moved into Settings out of the floating cluster, and emoji chrome retired in favor of consistent inline icons.
- **Scales gracefully.** Older note months collapse by default, long task lists scroll cleanly, and the earlier-results archive stays navigable as it grows.

## For PIs

- **Lab Overview.** A dedicated, curated landing page for principal investigators.
- **Required lab password.** A lab password is now set at folder setup and required at login, keeping your workspace private.

## Lab tools

- **Scientific calculator.** A general-purpose calculator joins the lab calculators (molarity, dilutions, serial dilutions, primer Tm, DNA/RNA, buffers), reachable from anywhere.
- **More accurate primer Tm.** Melting-temperature math now uses the nearest-neighbor (SantaLucia) model instead of a percent-GC estimate, and the primer Tm tool needs only the sequence.

## Open source

- **Built on open source.** A new in-app page credits the open-source projects ResearchOS is built on, paired with repository notice files, honoring both the spirit of the community and the attribution terms of the licenses we depend on.

## Other improvements

- Purchases gains a one-tap reorder row.
- Photo annotation popup polish and a cleaner image edit mode.
- Telegram polling is stable across multiple open tabs.
